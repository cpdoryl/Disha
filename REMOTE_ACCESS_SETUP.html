Quick Start

Open Browser (Any Device)

Connect to Session

Find "🖥️ Disha Development" with a green dot

Click to connect. ✅ You're in!

  • Ask follow-up questions

Troubleshooting

❓ Session Not Showing?

Check: Is terminal still running? Refresh browser. Log out and log back in. Check internet connection.

❌ Connection Dropped?

Fix: Restart terminal with remote control command. Refresh browser. Reconnect to session.

🐢 Slow Response?

Try: Check internet speed. Verify local machine isn't busy. Restart the session.

🔄 Changes Not Syncing?

Solution: Refresh browser. Wait 3-5 seconds. Verify file is saved locally. Restart session.

Status Indicators

🟢 GREEN dot: Session ONLINE & CONNECTED (ready to work)
🟡 YELLOW dot: Session CONNECTING (wait a moment)
🔴 RED dot: Session OFFLINE (terminal closed)
⚪ GRAY dot: Session INACTIVE (not running)

Security & Privacy

  • Your code stays on your local machine (not uploaded)
  • Browser is just a "viewing window" into local code
  • Only your account can connect to this session
  • Session ends when you close the terminal
  • No code stored in cloud, encrypted connection
  • Don't share session with others (they'd see all code)
  • Log out and close terminal when done
